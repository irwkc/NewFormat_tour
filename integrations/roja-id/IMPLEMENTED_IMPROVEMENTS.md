## 🔧 Как это работает

### При регистрации:
1. Frontend применяет предобработку изображения
2. Извлекается дескриптор лица
3. Backend дополнительно обрабатывает изображение (если модели есть)
4. Дескриптор сохраняется в базе как массив (готов к расширению)

### При входе:
1. Frontend применяет предобработку
2. Извлекается дескриптор лица
3. Backend сравнивает со всеми сохраненными дескрипторами
4. Используется минимальное расстояние
5. Применяется адаптивный порог (если доступен)
6. Обновляется история для будущих улучшений порога
