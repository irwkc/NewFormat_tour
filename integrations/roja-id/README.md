# Система аутентификации по лицу

Веб-приложение для аутентификации пользователей по лицу с **продвинутой защитой от спуфинга** (фото, видео, печатные фото). Все алгоритмы разработаны самостоятельно без использования платных сервисов.

## ⚡ Ключевые возможности

✅ **Продвинутый анализ изображений** - обнаружение печатных фото через анализ текстуры и освещения  
✅ **Многоуровневый Liveness Detection** - проверка естественности миганий, движений и временных паттернов  
✅ **База данных с статистикой** - отслеживание успешности аутентификации  
✅ **Серверная валидация** - все проверки выполняются на сервере  
✅ **Расширенный сбор данных** - история движений, миганий и последовательность кадров  

## Архитектура

### Frontend
- Захват видео с веб-камеры с высоким качеством
- Продвинутый Liveness Detection с отслеживанием:
  - История миганий с временными метками и интервалами
  - История движений головы с направлениями и скоростями
  - Последовательность кадров для анализа изменчивости
- Визуальный индикатор прогресса проверки
- Отправка расширенных данных на сервер

### Backend
- **Анализ изображений** (`imageAnalysis.js`):
  - Анализ текстуры (дисперсия Лапласиана, гистограмма, частотные паттерны)
  - Анализ освещения (направление света, вариации теней)
  - Обнаружение артефактов печати
- **Продвинутый Liveness Detection** (`advancedLivenessDetection.js`):
  - Временная согласованность (длительность, частота кадров)
  - Качество движения (разнообразие, естественность интервалов)
  - Паттерн мигания (ритм, вариативность)
  - Анализ последовательности кадров
- **База данных** (`database.js`):
  - JSON-based хранилище дескрипторов
  - Статистика аутентификации
  - Резервное копирование
- **Распознавание лиц** (`faceRecognition.js`):
  - Извлечение дескрипторов через face-api.js
  - Сравнение дескрипторов с евклидовым расстоянием

## Защита от спуфинга

### Реализованные методы защиты:

1. **От статичных фото:**
   - ✅ Анализ изменчивости между кадрами
   - ✅ Требование множественных движений (минимум 3)
   - ✅ Временные проверки (минимум 3 секунды)

2. **От печатных фото:**
   - ✅ Анализ текстуры (дисперсия Лапласиана, паттерны принтера)
   - ✅ Анализ гистограммы (характерные зубцы)
   - ✅ Проверка градиентов (неестественные переходы)
   - ✅ Анализ освещения (плоское освещение = фото)

3. **От видео:**
   - ✅ Проверка естественности миганий (ритм, вариативность)
   - ✅ Анализ ритма движений
   - ✅ Временная согласованность (частоты кадров)
   - ✅ Проверка разнообразия движений

4. **Временные проверки:**
   - ✅ Минимум 2 мигания с естественными интервалами
   - ✅ Минимум 3 движения головы в разные стороны
   - ✅ Минимальная длительность процесса (3+ секунды)
   - ✅ Максимальная длительность (защита от затягивания)

## Технологии

### Frontend
- HTML5 + JavaScript (Vanilla JS, без фреймворков)
- MediaDevices API для захвата видео
- **Face-api.js** (CDN) - обнаружение лиц и landmarks
- Canvas API для обработки изображений

### Backend
- **Node.js + Express** - серверная часть
- **face-api.js** - распознавание лиц (open-source)
- **Canvas** (node-canvas) - обработка изображений на сервере
- **JSON-база данных** - файловое хранилище дескрипторов
- **Собственные алгоритмы** - анализ текстуры, освещения, liveness detection

### Собственные алгоритмы (без платных сервисов):
- ✅ Анализ текстуры изображения (Variance of Laplacian, гистограмма)
- ✅ Анализ освещения (направление света, вариации теней)
- ✅ Временной анализ движений
- ✅ Анализ ритма миганий
- ✅ Проверка согласованности кадров

## Установка и запуск

См. инструкции в папках `frontend/` и `backend/`
