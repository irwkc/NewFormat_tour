# Инструкция по быстрому запуску

## Автоматическая установка

Я создал скрипт для автоматической установки всех необходимых компонентов. Выполните в терминале:

```bash
cd "/Users/irwkc/Developer/Roja id"
./setup.sh
```

Этот скрипт:
1. ✅ Установит Homebrew (если не установлен)
2. ✅ Установит Node.js и npm (если не установлены)
3. ✅ Установит все зависимости проекта
4. ✅ Подготовит проект к запуску

**Примечание:** При установке Homebrew потребуется ввести пароль администратора.

---

## Ручная установка

Если автоматический скрипт не подходит, выполните вручную:

### 1. Установка Homebrew
```bash
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/HEAD/install.sh)"
```

После установки добавьте Homebrew в PATH (для Apple Silicon Mac):
```bash
echo 'eval "$(/opt/homebrew/bin/brew shellenv)"' >> ~/.zprofile
eval "$(/opt/homebrew/bin/brew shellenv)"
```

### 2. Установка Node.js
```bash
brew install node
```

### 3. Установка зависимостей проекта
```bash
cd "/Users/irwkc/Developer/Roja id"
npm install
```

---

## Запуск сервера

После установки всех компонентов:

```bash
cd "/Users/irwkc/Developer/Roja id"
npm start
```

Сервер запустится на `http://localhost:3000`

## Открытие frontend

### Вариант 1: Просто открыть файл
Откройте файл `frontend/index.html` в браузере (двойной клик)

### Вариант 2: Локальный сервер
```bash
cd frontend
python3 -m http.server 8000
```
Затем откройте `http://localhost:8000`

## Использование

1. **Регистрация:**
   - Откройте вкладку "Регистрация"
   - Введите ID пользователя (например: `user1`)
   - Нажмите "Начать регистрацию"
   - Разрешите доступ к камере
   - Нажмите "Захватить лицо"

2. **Вход:**
   - Откройте вкладку "Вход"
   - Введите тот же ID пользователя
   - Нажмите "Начать вход"
   - Выполните проверку:
     - Мигните **не менее 2 раз**
     - Поверните голову **в разные стороны** (влево, вправо, вверх, вниз) - **минимум 3 движения**
     - Процесс займет около 3-5 секунд

## Если что-то не работает

### Ошибка "npm not found"
- Убедитесь, что Node.js установлен: `node --version`
- Перезапустите терминал после установки
- Проверьте PATH: `echo $PATH`

### Камера не работает
- Разрешите доступ к камере в настройках браузера
- Используйте Chrome или Firefox
- Убедитесь, что используется HTTPS или localhost

### Модели не загружаются
- Проверьте интернет-соединение (модели загружаются из CDN)
- Модели на frontend загружаются автоматически

### Ошибки при установке зависимостей
```bash
npm cache clean --force
npm install
```
