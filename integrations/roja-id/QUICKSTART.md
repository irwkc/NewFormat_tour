# Быстрый старт

## Шаг 1: Установка зависимостей

Убедитесь, что у вас установлен Node.js (версия 14 или выше):
```bash
node --version
npm --version
```

Если Node.js не установлен, скачайте с [nodejs.org](https://nodejs.org/)

Затем установите зависимости:
```bash
cd "/Users/irwkc/Developer/Roja id"
npm install
```

## Шаг 2: Настройка моделей face-api.js (для backend)

Модели загружаются автоматически из CDN на frontend, но для backend их нужно скачать.

**Вариант 1 (автоматический - через скрипт):**
```bash
cd backend/models
# Модели загрузятся автоматически при первом запуске сервера
```

**Вариант 2 (вручную):**
Создайте директорию `backend/models` и скачайте модели:
- `ssd_mobilenetv1_model-weights_manifest.json`
- `ssd_mobilenetv1_model-shard1`
- `face_landmark_68_model-weights_manifest.json`
- `face_landmark_68_model-shard1`
- `face_recognition_model-weights_manifest.json`
- `face_recognition_model-shard1`
- `face_expression_model-weights_manifest.json`
- `face_expression_model-shard1`

Или используйте только frontend-модели (они загружаются из CDN автоматически).

## Шаг 3: Запуск сервера

В терминале выполните:
```bash
cd "/Users/irwkc/Developer/Roja id"
npm start
```

Или для разработки с автоперезагрузкой:
```bash
npm run dev
```

Сервер запустится на `http://localhost:3000`

## Шаг 4: Открытие frontend

**Вариант 1: Просто открыть файл**
Откройте файл `frontend/index.html` в браузере (просто двойной клик)

**Вариант 2: Локальный веб-сервер (рекомендуется)**
```bash
cd frontend
python3 -m http.server 8000
```
Затем откройте `http://localhost:8000` в браузере

**Вариант 3: Использовать сервер Express**
Если настроить Express для раздачи статических файлов, frontend будет доступен через backend.

## Шаг 5: Использование

1. **Регистрация:**
   - Откройте вкладку "Регистрация"
   - Введите ID пользователя
   - Нажмите "Начать регистрацию"
   - Разрешите доступ к камере
   - Нажмите "Захватить лицо"

2. **Вход:**
   - Откройте вкладку "Вход"
   - Введите тот же ID пользователя
   - Нажмите "Начать вход"
   - Выполните проверку подлинности:
     - Мигните не менее 2 раз
     - Поверните голову в разные стороны (минимум 3 движения)
     - Процесс займет около 3-5 секунд

## Возможные проблемы

### Камера не работает
- Убедитесь, что браузер имеет доступ к камере
- Проверьте настройки приватности браузера
- Используйте HTTPS (для production) или localhost

### Модели не загружаются
- Проверьте интернет-соединение (для CDN моделей на frontend)
- Или скачайте модели локально в `backend/models/`

### Ошибки при установке зависимостей
- Убедитесь, что Node.js установлен
- Попробуйте: `npm cache clean --force` и затем `npm install`
- Проверьте версию Node.js: должна быть 14 или выше

### Ошибка "Cannot find module"
- Убедитесь, что выполнили `npm install`
- Проверьте, что находитесь в правильной директории

## API Endpoints

После запуска сервера доступны следующие endpoints:

- `POST /api/register` - Регистрация нового пользователя
- `POST /api/authenticate` - Аутентификация по лицу
- `GET /api/users` - Список всех пользователей
- `GET /api/users/:userId/stats` - Статистика пользователя
- `GET /api/stats` - Общая статистика системы
- `DELETE /api/users/:userId` - Удаление пользователя

## Логи и отладка

Все логи выводятся в консоль сервера. При ошибках проверьте:
- Консоль браузера (F12) для ошибок frontend
- Консоль сервера для ошибок backend
