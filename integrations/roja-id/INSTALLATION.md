# Инструкция по установке и настройке

## Предварительные требования

- Node.js (версия 14 или выше)
- npm или yarn
- Веб-камера для тестирования

## Установка

1. **Установите зависимости:**
```bash
npm install
```

2. **Создайте файл `.env`:**
```bash
cp .env.example .env
```

3. **Настройте модели face-api.js (для backend):**

Модели уже загружаются из CDN на frontend, но для backend нужно скачать их локально:

```bash
mkdir -p backend/models
cd backend/models

# Скачайте модели face-api.js:
# https://github.com/justadudewhohacks/face-api.js-models

# Или используйте следующую команду (если установлен wget):
wget https://raw.githubusercontent.com/justadudewhohacks/face-api.js-models/master/ssd_mobilenetv1/ssd_mobilenetv1_model-weights_manifest.json
wget https://raw.githubusercontent.com/justadewhohacks/face-api.js-models/master/ssd_mobilenetv1/ssd_mobilenetv1_model-shard1

# Аналогично для других моделей:
# - face_landmark_68_model
# - face_recognition_model
# - face_expression_model
```

**Альтернатива:** Используйте только frontend-модели и отправляйте уже извлеченные дескрипторы на backend.

4. **Запустите сервер:**
```bash
npm start
# или для разработки:
npm run dev
```

5. **Откройте frontend:**
Откройте файл `frontend/index.html` в браузере или настройте простой веб-сервер:

```bash
cd frontend
python3 -m http.server 8000
# или используйте любой другой веб-сервер
```

## Настройка API URL

Измените `API_URL` в файле `frontend/app.js` на адрес вашего сервера:

```javascript
const API_URL = 'http://localhost:3000'; // или ваш URL
```

## Использование

1. **Регистрация:**
   - Введите ID пользователя
   - Нажмите "Начать регистрацию"
   - Разрешите доступ к камере
   - Нажмите "Захватить лицо"

2. **Вход:**
   - Введите ID пользователя
   - Нажмите "Начать вход"
   - Выполните проверку подлинности (мигание, повороты головы)
   - Система автоматически проверит аутентификацию

## Защита от спуфинга

Система включает следующие методы защиты:

1. **Мигание глаз** - обнаружение естественного мигания
2. **Движение головы** - требование поворотов влево-вправо
3. **Временная последовательность** - минимальное время процесса (3+ секунды)
4. **Множественные кадры** - анализ нескольких кадров для проверки

## Улучшение защиты

Для более надежной защиты рассмотрите:

1. **3D-структура лица** - использование стерео-камер или depth sensors
2. **Анализ текстуры** - определение печати на бумаге
3. **Биометрические параметры** - комбинация с другими биометриями
4. **Облачные API** - использование AWS Rekognition, Azure Face API для более точного распознавания

## Проблемы и решения

**Камера не работает:**
- Проверьте разрешения браузера
- Убедитесь, что используется HTTPS (для некоторых браузеров)

**Модели не загружаются:**
- Проверьте интернет-соединение (для CDN)
- Или скачайте модели локально

**Низкая точность распознавания:**
- Улучшите освещение
- Убедитесь, что лицо хорошо видно
- Настройте порог сходства в `backend/server.js` (переменная `threshold`)
