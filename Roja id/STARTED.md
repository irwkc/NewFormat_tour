## 📝 Как использовать:

1. **Регистрация:**
   - Вкладка "Регистрация"
   - Введите ID (например: `user1`)
   - Нажмите "Начать регистрацию"
   - Разрешите доступ к камере
   - Нажмите "Захватить лицо"

2. **Вход:**
   - Вкладка "Вход"
   - Введите тот же ID
   - Нажмите "Начать вход"
   - Выполните проверку:
     - ✨ Мигните **не менее 2 раз**
     - 👆 Поверните голову **в разные стороны** - **минимум 3 движения**
     - ⏱️ Процесс займет **3-5 секунд**

## 🔧 Если нужно остановить серверы:

```bash
# Остановить backend (Ctrl+C в терминале где запущен)
lsof -ti:3000 | xargs kill

# Остановить frontend сервер
lsof -ti:8000 | xargs kill
```

## 📊 API Endpoints (http://localhost:3000):

- `GET /api/users` - Список пользователей
- `POST /api/register` - Регистрация
- `POST /api/authenticate` - Аутентификация
- `GET /api/users/:userId/stats` - Статистика пользователя
- `GET /api/stats` - Общая статистика
