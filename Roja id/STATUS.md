## 🚀 Как использовать:

### 1. Откройте frontend

**Вариант А (проще):**
Просто откройте файл `frontend/index.html` в браузере (двойной клик)

**Вариант Б (локальный сервер):**
```bash
cd frontend
python3 -m http.server 8000
```
Затем откройте `http://localhost:8000`

### 2. Регистрация пользователя

1. Откройте вкладку **"Регистрация"**
2. Введите ID пользователя (например: `user1`)
3. Нажмите **"Начать регистрацию"**
4. Разрешите доступ к камере
5. Нажмите **"Захватить лицо"**

### 3. Вход по лицу

1. Откройте вкладку **"Вход"**
2. Введите тот же ID пользователя
3. Нажмите **"Начать вход"**
4. Выполните проверку подлинности:
   - ✨ Мигните **не менее 2 раз**
   - 👆 Поверните голову **в разные стороны** (влево, вправо, вверх, вниз) - **минимум 3 движения**
   - ⏱️ Процесс займет около **3-5 секунд**

## 📊 API Endpoints

Все endpoints доступны на `http://localhost:3000`:

- `GET /api/users` - Список всех пользователей ✅
- `POST /api/register` - Регистрация нового пользователя
- `POST /api/authenticate` - Аутентификация по лицу
- `GET /api/users/:userId/stats` - Статистика пользователя
- `GET /api/stats` - Общая статистика системы
- `DELETE /api/users/:userId` - Удаление пользователя
